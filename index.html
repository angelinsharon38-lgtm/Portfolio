Current Project

Smart Accident Detection & Alert System

The Smart Accident Detection System is an embedded solution developed to detect vehicular collisions in real time using ESP32 and MPU6050 sensors. It continuously monitors acceleration and motion patterns to identify sudden impacts. Upon detection, the system retrieves GPS location data. The location is then transmitted to enable faster emergency response and improved road safety.
